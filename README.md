## Overview
Task: Tạo 1 website chat video 2 người, dùng công nghệ WebRTC và Websocket.

### Tóm tắt  bài làm.

Firebase:
- Auth.
- Room(Room, RoomCreater, check Room exist): Firestore.

Socket.io:
- Signaling.
- Gửi tin nhắn, file.
  - Không lưu lại, F5 lại sẽ xóa hết các tin nhắn và file đã nhận hoặc gửi.
